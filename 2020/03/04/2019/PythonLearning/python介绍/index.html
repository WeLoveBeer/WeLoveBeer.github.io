关于编译型语言和解释性语言

编译型语言诸如c,c++


而解释性语言一般先生成一个字节码，例如python和Java，然后交给虚拟机或者叫解释器去执行，每解释一句就执行一句

知乎上有人说的很有道理，编译型语言就像把菜都上齐了再吃，解释性语言就像吃火锅。

garbage collection垃圾回收，当失去最后一个引用时

python 程序的执行过程

首先把.py文件读入内存，然后检查是否有与.py文件对应的字节码文件如.pyc或.pyo,并检查他们的时间戳，如果没有过期就使用字节码，否则将.py文件parse成字节码，交给虚拟机去执行。
