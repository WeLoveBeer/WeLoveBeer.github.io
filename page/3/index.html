原来qt的layout也是可以自定义的.
qt原生有 QHBoxLayout, QVBoxLayout, QGridLayout, and QFormLayout这几个layout。
自定义layout就实现下面几个方法就行。

  • A data structure to store the items handled by the layout. Each item is a QLayoutItem. We will use a QVector in this example.
  • addItem(), how to add items to the layout.
  • setGeometry(), how to perform the layout.
  • sizeHint(), the preferred size of the layout.
  • itemAt(), how to iterate over the layout.
  • takeAt(), how to remove items from the layout.

5月最后一天

赶在5月的最后一天来一发日记。
不知不觉，这一年又快一半了，感觉自己的成长真的很慢，学习新的东西没灵性，自己的逻辑思维能力真的有限，比不了江涛他们。而且从长远来看，这种缓慢的成长又对自己很不利，只能说逆水行舟，不进则退。自己写的代码总是有问题很容易让人有自卑感，确实，让人感到累的不是繁重的工作，而是不快乐。
我又一次感到不快乐，感到孤独，千辛万苦等到周末，而周末又是这么无聊地度过，像是打球不开心的缘故，又是周五下班前发现的一个bug?

找到对象并不能让我开心，年薪百万也不能让我开心，会一万种技能不能让我开心；唯一能使我开心的是良好的人际关系，稳定的几个朋友，但是我没有，不是我不想，而是自己吧自己关了起来，即使别人表示好意，自己也去拒绝，是在自我保护。保护自己不会再像之前一样受伤，保护防御力低，攻击力为0的自己。总是在判断别人的行为有没有攻击行为。
长大的过程中，很少有人在我最需要保护的时候出现。能想的起来的那些人都是我很感激的人，在大学失恋的时候晓明饭桌上让我哭了出来，那是我走出失恋泥潭的转折点，我不止一次的想表达自己的感谢，但就是没有机会开口。而家里的长辈们只能说擅长养孩子，跟养阿猫阿狗一样，从来也没进入我的心里，也很少能感到亲近，这或许是自己亲密关系缺失的一个重要原因。小时候偷家里钱，被发现钻到被窝哭红了眼，在家人审判声中丧失尊严的时候自己默默承受；在上学的时候受欺负的时候自己默默承受；在恋爱时自己吃着面啪嗒啪嗒往碗里掉眼泪的时候自己默默承受。当个好人太难了，这让我成了一个记仇的人，没有发泄出去的怨念让我成为一个记仇的人，我记得践踏我尊严的那个场景，我记得受同学欺负时候老师没有站在我这边的时候的样子，我记得当住在自己心里的人狠狠地捅了自己一下的那种感觉。这是一个下行循环，而我却一直在这个循环里转悠。让我最受伤的是亲密关系，让自己最开心的也是亲密关系。
Still pretending .Pretending to be a good son; Pretending to be a good friend; Pretending to be a good employee.

我只是感到很孤独。

关于qt事件的学习

使用widget本身的事件

  1. actionEvent(QActionEvent *event)

    每当添加删除action的时候触发，关于Action:

    In applications many common commands can be invoked via menus, toolbar buttons, and keyboard shortcuts. Since the user expects each command to be performed in the same way, regardless of the user interface used, it is useful to represent each command as an action.

  2. changeEvent(QEvent *event)

    widget change的时候触发，包括以下change:

    Change events include: QEvent::ToolBarChange, QEvent::ActivationChange, QEvent::EnabledChange, QEvent::FontChange, QEvent::StyleChange, QEvent::PaletteChange, QEvent::WindowTitleChange, QEvent::IconTextChange, QEvent::ModifiedChange, QEvent::MouseTrackingChange, QEvent::ParentChange, QEvent::WindowStateChange, QEvent::LanguageChange, QEvent::LocaleChange, QEvent::LayoutDirectionChange, QEvent::ReadOnlyChange

  3. closeEvent(QCloseEvent *event)

  4. contextMenuEvent(QContextMenuEvent *event)

    右键菜单又叫context menu,上下文菜单

  5. dragEnterEvent(QDragEnterEvent *event)

  6. dragLeaveEvent(QDragLeaveEvent *event)

  7. dragMoveEvent(QDragMoveEvent *event)

    当有一个drag 在in progress的时候才会触发

  8. dropEvent(QDropEvent *event)

  9. enterEvent(QEvent *event)

  10. leaveEvent(QEvent* event)

    鼠标cursor 进入触发

  11. focusInEvent(QFocusEvent *event)

  12. focusOutEvent(QFocusEvent *event)

  13. hideEvent(QHideEvent *event)

  14. showEvent(QHideEvent *event)

  15. keyPressEvent(QKeyEvent *event)

  16. keyReleaseEvent(QKeyEvent *event)

  17. mouseDoubleClickEvent(QMouseEvent *event)
    mouseMoveEvent(QMouseEvent *event)
    mousePressEvent(QMouseEvent *event)
    mouseReleaseEvent(QMouseEvent *event)

  18. inputMethodEvent(QInputMethodEvent *event)
    输入文字到widget中所处理的

  19. moveEvent(QMoveEvent *event)
    widget被移动的事件

  20. nativeEvent(const QByteArray &eventType, void *message, long *result)
    可以捕获windows原生MSG

  21. paintEvent(QPaintEvent *event)

  22. resizeEvent(QResizeEvent *event)

  23. showEvent(QShowEvent *event)

  24. tabletEvent(QTabletEvent *event)

五四青年节

早上弹了很长时间的钢琴，发现弹钢琴是个很好的打发时间的方式，弹钢琴的时候时间过的很快。

下午打了很爽的篮球，篮球还是自己最喜欢的兴趣爱好，今天应该算是很好的篮球状态了，自己有投篮，和队友有配合，这大概是打篮球最好的状态了。

有点小不爽的是撞到了一个小孩，还把他撞到鼻子流血、嘴唇流血，我们已经跟他们家人说过他在可能受伤，但是大人还是让小孩在旁边玩，小孩是有很大可能受伤的。

我过去道歉，最后大人也没说什么，只是一起打球的一个人对我有意见，估计这个人是这个小孩的什么亲戚吧，咨询了警察兄弟，儿童是重点关注对象，两个人都是有责任的，最多赔点钱，也没啥大问题，这个家人应该长记性了。

五一假期做了很多事情，虽然没出去玩，因为没人一起，，但是自己确实学了很多，把自己的博客美化了下；学了obs录视频，做直播…

今天是五四青年节，1919年的今天，许多人献出了鲜血，中国忘不了这段耻辱的历史，假如自己生活在那个时代，是否有勇气为国奉献？今天这个时代没有销烟，没有不平等条约，但是为自己脚下这片热土，为自己期望生存的环境而战的五四精神不能忘，愿自己心中有火，眼里有光。

B站的后浪刷爆了朋友圈，我们这一代不同上一代，有多种选择，有太多生活方式，无论如何都是不可能饿死的，路有很多条，每条路有很多走法，属于我们青年的时代。
