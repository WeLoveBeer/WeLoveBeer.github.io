又一年国庆

本来想等一下纯黑的留海回消息，但是她并没回，那就是没有缘分，就不要浪费感情了，反正也没见到真人，也没买到G3110的票，也没送出月饼。

昨晚打游戏到3点钟，看着听着开黑的网友开心地聊天，自己感觉好孤单，拿游戏去麻痹自己。

今晚看了《我和我的家乡》，带给我很多思考，我觉得自己独处的时间很容易陷入下行循环，就是要找一些能带给自己思考的事情去做。影片里好多人都为自己家乡做出了很多贡献，我感觉自己并没有真正地发挥自己的光和热，只是在闷头写代码，给网易，给杭州贡献力量，但是养育自己的家乡还是那么落后，而自己却没有任何回报，自己像是没有真正地活着，或许push 自己so hard了，但是我并不喜欢现在的自己，不太喜欢平庸的工作，碌碌无为；里面还有一段马亮为家乡画油画，然后现实就是画里面的内容，我真的感到了艺术带来的魅力，算是一种peak experience吧；

什么样的生命是更有价值的呢，是终其一生为在西北抗沙的人民，实现沙漠变绿洲的伟业更有价值，还是当一名人民教师，培养出很多能发光发热的生命更有价值，还是为一个企业干一辈子，在别人看来光鲜亮丽，然后满怀遗憾地老去？

做有意义的事情，做发光发热的人！

关于编译型语言和解释性语言

编译型语言诸如c,c++


而解释性语言一般先生成一个字节码，例如python和Java，然后交给虚拟机或者叫解释器去执行，每解释一句就执行一句

知乎上有人说的很有道理，编译型语言就像把菜都上齐了再吃，解释性语言就像吃火锅。

python

dynamically typed
strongly typed
Everything in Python is an object
first-class object(一般来说object可以被用来当做参数传递，或者可以值返回)
garbage collection垃圾回收，当失去最后一个引用时

NoneType object - None
一些类型是不可变得（immutable）：numbers, strings, and tuples
一些类型是可变的：lists and dictionaries

多态

子类重写父类的虚函数，返回值类型不能变，如果变了编译器报错；
参数类型和个数不能变，不然使用基类指针将不能调用派生类的函数，这个函数将只能被派生类对象或指针调用。

当class内含有至少一个虚函数时才声明析构函数为虚函数；
虚表指针地址不和栈空间分配的成员变量连续

栈空间分配的内存不能使用delete释放，如果这样会报_block_type_is_valid(phead- nblockuse)错误；
字节对齐：
公式1:前面的地址必须是后面的地址正数倍,不是就补齐
公式2:整个Struct的地址必须是最大字节的整数倍

平庸的一天

早上闹钟没响！！！

今天早上第一次没听见闹钟响，或者是闹钟压根就没响，没有正常而规律的早起让我十分慌张，打了个车去公司，不好没迟到。生活不会告诉你该怎么办，只会推着你向前走。

工作即是工作，能带来很少的成就感，说实话现在感到十分地迷茫，总感觉在等一个人，在等一件事，在等某一个时刻，仿佛只要这个时刻到来，自己就能华丽地蜕变。但是自己想要什么都不知道，又怎么能期望蜕变出来新的自己会是自己满意的呢?

心理学说幸福感的获得有三个因素：

满足归属需要，并与别处两种人类需要（自主感和能力感）保持平衡，通常就会带来深深的幸福感。

看来现在三样每一样都不如人意，没有归属感：前段时间有个同学说要在杭州定居了，问我之后的打算，我很犹豫，在杭州确实没归属感，没有自己的房子，没有知心的朋友，没有能让自己内心不满与无助说出来的时刻。唯一能带来快感的恐怕只有篮球了；自主感就是自由吧，上班不自由就是最大的不自由；能力感上，虽然现在能胜任当前的工作，但是感觉现在进步的很慢，没有考研时候的那种积极向上的动力，只有盼着假期的心。

投资未来的人是忠于现实的人

自己也在想怎么投资未来，自己的未来是什么样子的，现在想想，自己现在确实没有能拿出手的能力，工作能力一般般，社交能力，情商一般般，智商一般般，口才一般般，奔三的人了，脸皮还如纸一样薄。至少现在已经开始思考未来，开始理财。。。

其实人成长的过程也是在不断实现更大程度的自由，能力越强，可选择的路就越多，，反过来，有越多选项的人，能力也就越强，越能感到自由；就像自己从事技术工作的，能力覆盖的越广，未来越坦荡，只是现在自己的选项很少。

可能是自己想的太多吧，或者是要求的太多，或者是自己不懂感恩，总觉得达不到预期。
天下大事必作于细，天下难事必作于易。时间总还是有的，时间对于每个人来说都是公平的，决定未来的就是现在。

And not, when I have to die, discover that I had not lived!

2019/PythonLearning/python string(2)

python string（2）

单双引号是一样的，为了减少\的使用。

可以构造类似 s = ‘a\nb\tc’这样的字符串，里面可以加入转义字符。

  • 每一个字符串都是Unicode string。
  • 纯字符，前面加r或R：

    myfile = open(r’C:\new\text.txt’, ‘w’)

  • 三引号用来输入多行文本
  • string切片，是左闭右开的，正负都是有界限的，越界将发生out of range错误
  • s1=s2是浅拷贝，s1=s2[:]是深拷贝，这个特性对immutable 作用不大，但是多list这样mutable作用大。
  • X[i:j:k]， “extract all the items in X, from offset i through j-1 by k.”，S[::-1]逆序输出

一些字符串函数

  • count(sub[, start[, end]])，获取sub的个数
  • find(sub[, start[, end]])
  • replace( old, new[, count]) 替换
  • partition( sep)返回一个三元组
  • join( seq) Return a string which is the concatenation of the strings in the sequence seq

    ‘1, 2, 3’.split(‘, ‘)
    [‘1’, ‘2’, ‘3’]

    import re

    split with semicolon, comma, space

    str = ‘Life; a walking, shadow’
    str1 = re.split(‘;|,| ‘, str)
    str1
    [‘Life’, ‘’, ‘a’, ‘walking’, ‘’, ‘shadow’]

    split with semicolon+space, comma+space, space

    str2 = re.split(‘; |, | ‘,str)
    str2
    [‘Life’, ‘a’, ‘walking’, ‘shadow’]
